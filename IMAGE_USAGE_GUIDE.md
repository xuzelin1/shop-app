# 图片资源使用指南

## ⚠️ 版权说明

直接使用徕卡和哈苏官网的产品图片可能涉及版权问题。对于商业项目，建议使用以下合法方案：

## 💡 推荐方案

### 1. 占位图片服务（开发阶段）

#### Placeholder.com
```
https://via.placeholder.com/800x1000/2C2C2C/FFFFFF?text=Leica+M11
```

#### Placehold.co
```
https://placehold.co/800x1000/1a1a1a/white/png?text=Leica+M11+Case
```

#### Lorem Picsum (随机相机图片)
```
https://picsum.photos/800/1000?random=1
```

### 2. 免费图库（合法商用）

#### Unsplash
- 网址: https://unsplash.com
- 关键词: "camera leather", "leica camera", "hasselblad", "camera case"
- 许可: 免费商用

#### Pexels
- 网址: https://pexels.com
- 关键词: "vintage camera", "camera accessories", "leather goods"
- 许可: 免费商用

#### Pixabay
- 网址: https://pixabay.com
- 关键词: "camera", "photography equipment"
- 许可: 免费商用

### 3. 自制图片方案

#### 选项 A: 购买素材
- Shutterstock
- Adobe Stock
- Getty Images

#### 选项 B: 自己拍摄
- 购买样品自行拍摄
- 聘请摄影师
- 使用手机拍摄（适合 MVP）

### 4. AI 生成图片

使用 AI 工具生成产品图片：
- DALL-E 3
- Midjourney
- Stable Diffusion

提示词示例：
```
"Professional product photography of a black leather camera case 
for Leica M11, premium Italian leather, studio lighting, 
white background, 4K quality"
```

## 🛠️ 实现建议

### 开发阶段
1. 使用 placeholder 图片
2. 使用免费图库的通用相机图片
3. 使用 AI 生成的图片

### 生产阶段
1. 购买正版图片素材
2. 自己拍摄产品照片
3. 获得品牌方授权

## 📝 使用徕卡/哈苏官方图片的合法途径

### 1. 官方合作
- 联系徕卡/哈苏官方
- 申请经销商/合作伙伴资格
- 获得图片使用授权

### 2. 品牌大使计划
- 申请成为品牌推广者
- 获得官方素材使用权限

### 3. 新闻/评测用途
- 标注来源和版权信息
- 仅用于新闻报道或产品评测
- 遵守合理使用原则

## 🎨 当前项目建议

对于你的电商演示项目，我建议：

1. **短期方案**: 使用占位图片 + 免费图库
2. **中期方案**: 使用 AI 生成专业产品图
3. **长期方案**: 拍摄真实产品照片或购买授权

## 🔍 相关资源

## ⚖️ 法律建议

**切勿**:
- 直接下载品牌官网图片
- 截图或复制竞品网站图片
- 使用水印图片后去除水印

**应该**:
- 使用有明确许可的图片
- 标注图片来源
- 遵守使用条款
