### 4. 设计特点

#### 色彩方案
- 主色：白色、黑色、灰色
- 强调色：蓝色 (#2563eb)
- 辅助色：渐变紫蓝

#### 字体层级
- 大标题：5xl - 6xl (48-60px)
- 中标题：3xl - 4xl (30-36px)
- 小标题：2xl - 3xl (24-30px)
- 正文：base - xl (16-20px)

#### 圆角设计
- 大区块：3xl (24px)
- 卡片：2xl (16px)
- 按钮：full (完全圆角)

#### 间距
- 页面边距：max-w-[980px]（Apple 官网标准）
- 区块间距：py-12 到 py-20
- 元素间距：gap-4 到 gap-12

## 🎨 可自定义部分

如需调整：
1. **商品数据**：编辑 `src/data/products.ts`
2. **颜色主题**：修改 `tailwind.config.js`
3. **布局宽度**：搜索 `max-w-[980px]` 并修改
4. **图片**：替换 Unsplash URL

## 📝 注意事项

1. 图片使用 Unsplash 动态链接，需要网络连接
2. Next.js Image 组件已配置支持 `images.unsplash.com`
3. 所有页面均采用服务端组件，性能优秀
4. 响应式设计已适配移动端、平板、桌面
