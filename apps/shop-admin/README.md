# 后台管理系统

电商平台的后台管理系统，基于 React + Vite 开发。

## 功能特性

- 📊 数据统计仪表盘
- 📦 商品管理
- 📋 订单管理
- 👥 用户管理
- 👨‍💼 权限控制

## 开发指南

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览生产版本
pnpm preview
```

## 默认登录

开发环境下，使用任意用户名和密码即可登录。

生产环境需要实现真实的认证逻辑。

## 技术栈

- React 18
- Vite
- TypeScript
- Ant Design 5
- React Router v6
- Zustand
- Axios
