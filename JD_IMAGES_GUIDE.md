## 🔍 如何从京东获取真实图片URL

### 方法 1: 浏览器开发者工具（推荐）

1. **访问京东产品页面**
   ```
   示例：搜索"徕卡M11" 或 "哈苏X2D"
   https://search.jd.com/Search?keyword=徕卡M11
   ```

2. **打开开发者工具**
   - Windows/Linux: 按 `F12` 或 `Ctrl+Shift+I`
   - Mac: 按 `Cmd+Option+I`

3. **切换到 Network 标签**
   - 点击顶部的 "Network" (网络) 标签
   - 在筛选器中输入 "360buyimg" 或点击 "Img" 筛选图片

4. **刷新页面并查找图片**
   - 刷新页面 (F5)
   - 查看网络请求列表
   - 找到产品主图（通常是最大的图片）

5. **复制图片URL**
   - 点击图片请求
   - 右键 → "Copy" → "Copy URL" 或 "Copy link address"
   - 粘贴URL到记事本

**京东图片URL格式示例**:
```
https://img14.360buyimg.com/n1/jfs/t1/123456/12/12345/123456/5f6a1234E1234abcd/1234567890abcdef.jpg
```

### 方法 2: 直接在页面获取（简单）

1. **右键点击产品图片**
2. **选择 "在新标签页中打开图片"**
3. **复制地址栏的URL**
4. **粘贴到配置文件中**

### 方法 3: 检查元素

1. **右键点击产品图片**
2. **选择 "检查" 或 "审查元素"**
3. **在弹出的HTML代码中找到 `<img>` 标签**
4. **查看 `src` 属性的值（就是图片URL）**
5. **复制完整的URL**

## 🔧 京东图片URL详解

### URL结构
```
https://img{N}.360buyimg.com/{size}/jfs/t1/{随机路径}/{文件名}.jpg

组成部分：
- img{N}: CDN服务器编号 (10-14)
- {size}: 图片尺寸
  - n0: 原图
  - n1: 800x800
  - n4: 450x450
  - n5: 350x350
- jfs/t1/...: 京东文件系统路径
```

### 示例真实URL
```
https://img14.360buyimg.com/n1/jfs/t1/196407/23/28891/124011/6364c8a2E9b5e3d1f/4c8a0d3e8f7b2c1a.jpg
```
