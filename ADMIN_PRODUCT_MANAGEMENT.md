# 管理后台 - 商品管理功能

## 📦 功能概述

已为管理后台添加完整的商品管理功能，包括商品列表、添加商品和编辑商品页面。

## 🎯 实现的功能

### 1. 商品列表页面 (`/products`)

#### 功能特性：
- ✅ **商品展示**：表格形式展示所有商品
- ✅ **商品图片**：缩略图预览（60x60）
- ✅ **详细信息**：品牌、型号、分类、价格、库存
- ✅ **状态标签**：
  - 上架/下架状态
  - 推荐标记
  - 库存警告（<10件显示橙色，0件显示红色）
- ✅ **价格展示**：显示售价和原价（带删除线）
- ✅ **操作按钮**：
  - 编辑商品
  - 删除商品（带确认对话框）
  - 添加新商品
- ✅ **分页功能**：支持自定义每页显示数量
- ✅ **响应式布局**：自动适配不同屏幕尺寸

#### 表格列：
| 列名 | 描述 |
|------|------|
| 图片 | 商品缩略图（60x60） |
| 商品名称 | 名称 + 品牌型号 |
| 分类 | 产品分类 |
| 价格 | 售价 + 原价 |
| 库存 | 库存数量（带颜色标签） |
| 状态 | 上架/下架 + 推荐标记 |
| 操作 | 编辑/删除按钮 |

#### 模拟数据：
```typescript

- 哈苏 X2D 100C 专业皮套
- 徕卡 Q3 轻量半身套
- 徕卡 M10-P 经典皮套
- 哈苏 X1D II 50C 皮套
```

### 2. 商品表单页面 (`/products/new` 和 `/products/edit/:id`)

#### 功能特性：
- ✅ **双模式**：支持创建新商品和编辑现有商品
- ✅ **完整字段**：覆盖客户端所需的所有商品信息
- ✅ **表单验证**：必填字段验证
- ✅ **动态表单**：支持添加/删除规格、材料、特性
- ✅ **实时预览**：即时反馈表单状态
- ✅ **返回导航**：快速返回商品列表

#### 表单分组：

##### 1️⃣ 基本信息
- **商品名称** *（必填）
- **商品标语**（选填）
- **品牌** *（必填，下拉选择：Leica/Hasselblad）
- **型号** *（必填）
- **分类** *（必填，预设6个分类）
- **库存数量** *（必填）

##### 2️⃣ 价格信息
- **售价（¥）** *（必填）
- **原价（¥）**（选填，用于显示折扣）

##### 3️⃣ 商品描述
- **详细描述** *（必填，多行文本，最多1000字）

##### 4️⃣ 颜色选项
- **可选颜色** *（必填，多选/自定义）
- 预设颜色：经典黑、复古棕、大地褐、深海蓝、橄榄绿、酒红色、烟灰色、卡其色

##### 5️⃣ 产品规格
- **规格列表**（动态添加/删除）
- 示例：`意大利进口头层牛皮`、`手工缝制，精密裁剪`

##### 6️⃣ 材料清单
- **材料列表**（动态添加/删除）
- 示例：`意大利头层牛皮`、`日本YKK拉链`

##### 7️⃣ 产品特性
- **特性列表**（动态添加/删除）
- 每个特性包含：
  - 特性标题（如：`顶级材质`）
  - 特性描述（如：`精选意大利头层牛皮，手感细腻`）

##### 8️⃣ 产品尺寸（选填）
- 长度（mm）
- 宽度（mm）
- 高度（mm）
- 重量（g）

##### 9️⃣ 商品图片
- **图片链接**（多选，支持URL输入）
- 支持：京东CDN、Unsplash等

##### 🔟 商品状态
- **上架状态**（开关）：已上架/已下架
- **首页推荐**（开关）：是/否
- **新品标记**（开关）：是/否

## 💾 数据结构

### Product 接口（完整版）

```typescript
interface Product {
  // 基本信息
  id: string
  name: string                    // 商品名称
  slogan?: string                 // 商品标语
  brand: string                   // 品牌
  model: string                   // 型号
  category: string                // 分类
  
  // 价格信息
  price: number                   // 售价
  originalPrice?: number          // 原价
  
  // 库存
  stock: number                   // 库存数量
  
  // 描述信息
  description: string             // 详细描述
  colors: string[]                // 可选颜色
  specs: string[]                 // 产品规格
  materials: string[]             // 材料清单
  features: Array<{               // 产品特性
    title: string
    description: string
  }>
  
  // 尺寸（选填）
  dimensions?: {
    length: number                // 长度（mm）
    width: number                 // 宽度（mm）
    height: number                // 高度（mm）
    weight: number                // 重量（g）
  }
  
  // 图片
  images: string[]                // 图片URL数组
  
  // 状态
  status: 'active' | 'inactive'   // 上架状态
  isFeatured?: boolean            // 首页推荐
  isNew?: boolean                 // 新品标记
}
```

## 🚀 使用指南

### 添加新商品
1. 点击商品列表页的 **"添加商品"** 按钮
2. 填写所有必填字段（带 * 标记）
3. 可选填写其他信息
4. 点击 **"创建商品"** 保存

### 编辑商品
1. 在商品列表中点击对应商品的 **"编辑"** 按钮
2. 修改需要更改的字段
3. 点击 **"保存更改"** 更新

### 删除商品
1. 在商品列表中点击对应商品的 **"删除"** 按钮
2. 确认删除操作
3. 商品将被永久删除
